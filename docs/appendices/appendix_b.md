# Додаток Б

## Тест-кейси програмного комплексу

Таблиця Б.1. – Тест-кейси програмного комплексу

| Назва тест-кейсу | Алгоритм дій | Очікуваний результат |
|---|---|---|
| Обчислення ваг критеріїв | 1) Обрати профіль ОПР<br>2) Заповнити верхню трикутну частину матриці попарних порівнянь оцінками шкали Сааті<br>3) Натиснути обчислення ваг | Обчислено нормований вектор ваг критеріїв із одиничною сумою, відображений стовпчиковою діаграмою на кроці «Ваги» |
| Автозаповнення оберненими оцінками | Ввести оцінку в клітинку верхнього трикутника матриці попарних порівнянь | Симетрична клітинка нижнього трикутника автоматично набуває оберненого трикутного нечіткого числа |
| Контроль узгодженості суджень | 1) Задати матрицю з порушенням транзитивності суджень ($CR > 0{,}10$)<br>2) Запустити обчислення ваг | Система виводить повідомлення про неузгодженість і не обчислює ваги до виправлення матриці |
| Блокування недоступних кроків | У новому сеансі спробувати відкрити крок «Ранжування» до обчислення ваг | Крок недоступний, перехід можливий лише після обчислення ваг |
| Збереження стану кроків | Перейти на інший крок майстра і повернутися назад | Раніше введені матриця і параметри збережено, дані не втрачено |
| Ранжування локацій | 1) Після обчислення ваг перейти на крок «Ранжування»<br>2) Запустити ранжування | Таблиця локацій-кандидатів, упорядкована за спаданням коефіцієнта близькості $C_i^*$, зі значеннями відстаней $S_i^+$ і $S_i^-$ |
| Картографічне подання ранжування | 1) На кроці «Ранжування» відкрити карту<br>2) Розгорнути її на весь екран | Локації позначено маркерами з кольором за рангом, карта коректно відкривається в повноекранному режимі |
| Перегляд деталей локації | На карті натиснути маркер локації | Показано спливну картку зі значеннями критеріїв і рангом локації |
| Аналіз чутливості | 1) Перейти на крок «Чутливість»<br>2) Задати кількість ітерацій і амплітуду збурення ваг<br>3) Запустити аналіз | Побудовано індекси прийнятності рангів $p_i(k)$, гістограму розподілу коефіцієнтів близькості та графік збіжності середнього |
| Обмеження параметрів чутливості | Задавати кількість ітерацій і збурення повзунками | Значення обмежуються допустимим діапазоном, вихід за межі неможливий |
| Довірчі інтервали найкращих локацій | Переглянути результати чутливості для топ-локацій | Показано довірчі інтервали $C_i^*$ за перцентилями 2,5 і 97,5 % для трьох найкращих локацій |
| Порівняння профілів ОПР | Відкрити крок «Порівняння профілів» | Таблиця розбіжностей рангів двох стандартних профілів, згрупована діаграма та значення коефіцієнта рангової кореляції Спірмена |
| Повторне обчислення порівняння | На кроці «Порівняння профілів» натиснути «Оновити» | Результат порівняння перераховується заново |
| Скидання сеансу при зміні профілю | Змінити профіль ОПР після обчислення ваг і ранжування | Ваги і ранжування скидаються, майстер повертається на крок «Профіль і матриця» |
| Експорт ранжування | У формі ранжування обрати експорт у форматі CSV або JSON | Збережено файл із полями локацій, рангів і коефіцієнтів близькості |
| Експорт діаграм | На діаграмі ваг або чутливості обрати експорт | Збережено зображення діаграми у форматі PNG або SVG |
