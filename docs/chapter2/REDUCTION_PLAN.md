## Принципи скорочення

| Жертвуємо | Зберігаємо |
|---|---|
| Преамбульні абзаци «у цьому підрозділі ми розглянемо…» | Першу тезу одним реченням |
| Розгорнуті описи UML-нотацій | Семантику діаграми у предметних термінах |
| Окремі абзаци на кожен елемент діаграми | Список елементів з 1-рядковим описом кожного |
| 4-аргументні обґрунтування проєктних рішень | 1–2 ключові аргументи з прив'язкою до 1.3 |
| Завершальні «таким чином, ми побачили, що…» | Останнє речення-перехід до наступного підрозділу |
| Дублювання тез у різних підрозділах | Хрест-посилання («див. 2.X.Y») |
