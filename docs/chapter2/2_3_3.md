### 2.3.3. Алгоритм аналізу чутливості Монте-Карло

Алгоритм реалізує формули (1.15)–(1.17) підрозділу 1.2.4. Вхід: матриця рішень $X$, нормалізований вектор ваг $W$, параметри кількості ітерацій $N$, амплітуди збурення $\delta$ та глибини стабільності $K_{\max}$. Вихід: матриця частот стабільності $P = [p_i(k)]$ та 95 %-ві довірчі інтервали $C_i^*$ для топ-3 локацій. Алгоритм виконується у три фази:

1. **Ініціалізація.** Обнулення матриці лічильників $P$; створення порожніх колекторів значень $C_i^*$ для всіх альтернатив; фіксація насіння генератора для відтворюваності.
2. **Стохастичний цикл.** На кожній ітерації: (а) генерування рівномірних збурень $\varepsilon_j \sim U(-\delta, +\delta)$; (б) обчислення збуреного вектора ваг за (1.15); (в) ренормалізація за (1.16); (г) виклик алгоритму TOPSIS (підрозд. 2.3.2); (д) оновлення лічильників $P$ і колекторів $C^*$.
3. **Агрегація.** Обчислення частот стабільності $p_i(k)$ за (1.17); визначення топ-3 локацій за середніми значеннями $C^*$; побудова 95 %-вих довірчих інтервалів методом percentile bootstrap.

Складність алгоритму – $O(N \cdot n \cdot (m + \log n + K_{\max}))$. Порівняння асимптотичної складності трьох алгоритмів наведено у табл. 2.3.

Таблиця 2.3. – Порівняльна оцінка обчислювальної складності алгоритмів

| Алгоритм | Асимптотична складність |
|---|---|
| Fuzzy AHP (підрозд. 2.3.1) | $O(m^3)$ |
| TOPSIS (підрозд. 2.3.2) | $O(nm + n \log n)$ |
| Аналіз чутливості МК (підрозд. 2.3.3) | $O(N \cdot n \cdot (m + \log n + K_{\max}))$ |

Статистичну коректність результатів забезпечують симетричність розподілу збурень $U(-\delta, +\delta)$, що гарантує незсуненість $E[\tilde{W}] = W$, та ренормалізація збуреного вектора, що підтримує умову $\sum_j \tilde{w}_j = 1$. Довірчі інтервали обчислюються методом percentile bootstrap, що є адекватним для асиметричних розподілів $C^*$ поблизу меж стабільності ранжування.
