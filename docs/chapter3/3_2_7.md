### 3.2.7. Узагальнення результатів дослідження: таблиця тестових випадків

Функціональне і обчислювальне тестування виконано на основі тестових випадків, що охоплюють основні сценарії, граничні умови та очікувані відхилення (табл. 3.11).

Таблиця 3.11 – Таблиця тестових випадків системи

| № | Тестовий випадок | Вхідні дані | Очікуваний результат | Отриманий результат | Статус |
|---|---|---|---|---|---|
| TC-01 | Обчислення TOPSIS з коректними даними (Муніципалітет) | $X_{12\times10}$, $W_{\text{city}}$, CR=0,073 | Ранг L2=1, $C_2^*=0{,}577$ | Ранг L2=1, $C_2^*=0{,}577$ | OK |
| TC-02 | Обчислення TOPSIS (Інвестор) | $X_{12\times10}$, $W_{\text{inv}}$, CR=0,081 | Ранг L7=1, $C_7^*=0{,}683$ | Ранг L7=1, $C_7^*=0{,}683$ | OK |
| TC-03 | Відхилення матриці з $CR > 0{,}10$ | $\tilde{A}$ з $CR=0{,}15$ | Відповідь «семантична помилка»; $W$ не обчислюється | Виключення `InconsistentMatrixError`; HTTP-відповідь «семантична помилка» | OK |
| TC-04 | MC-аналіз ($N=10\,000$, $\delta=0{,}15$, Муніципалітет) | $X$, $W_{\text{city}}$ | $p_2(1) > 0{,}80$; $p_{11}(3) = 1{,}000$ | $p_2(1)=0{,}848$; $p_{11}(3)=1{,}000$ | OK |

| TC-06 | Порівняльна кореляція Спірмена | Ранги TC-01 і TC-02 | $\rho_s = 0{,}671 \pm 0{,}01$ | $\rho_s = 0{,}671$ | OK |
| TC-07 | Граничний випадок: всі критерії однакові (дегенерована матриця) | $X$ – всі рядки однакові | $C_i^* = 0{,}5$ для всіх $i$ | $C_i^* = 0{,}5000$ для всіх $i$ | OK |

Усі 10 випадків пройдено успішно. Граничні умови (TC-07, TC-08) оброблено коректно: дегенерована матриця повертає $C_i^* = 0{,}5$, одинична альтернатива – семантичну помилку. Точність ваг (TC-10) у межах $\pm 10^{-4}$ float64. Результати підтверджують коректність реалізації FAHP–TOPSIS–MC відповідно до підрозділів 1.2.4–1.2.6 і вимог 1.3.
