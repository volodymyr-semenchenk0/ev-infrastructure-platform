### 2.2. Orchestration <-> Chapter 2 / Appendix A

| Discrepancy | Fix | Commit |
|---|---|---|
| `ConfidenceInterval.low/high` instead of `lower/upper`. | Renamed in Pydantic schema and consumers. | `8f03256` |
| `stability_matrix` shape `{location_name: [p(rank=1), …]}` instead of `{locationId: {1, 3, 5}}`. | Service builds the location-id keyed top-k map from `top_k_freq`. | `8f03256` |
| CIs were emitted for all locations instead of top-3 by mean C*. | Service truncates to `TOP_N_FOR_CONFIDENCE_INTERVALS = 3`. | `8f03256` |

| Decision-matrix table named `location_criterion_values`; entity is `CriterionValue` per Appendix A.4. | Table renamed via Alembic 0003; ORM class renamed; all callers updated. | `702c265` |
| `SensitivityRepository.create` typing carried the old `dict[str, list[float]]`. | Updated to `dict[str, dict[str, float]]`. | `a7e5be4` |
