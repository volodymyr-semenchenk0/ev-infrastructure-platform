# Reference: Bibliography and Appendices

Read this reference when the user works on the bibliography list (СПИСОК ВИКОРИСТАНИХ ДЖЕРЕЛ) or on any aspect of appendices.

## Bibliography (СПИСОК ВИКОРИСТАНИХ ДЖЕРЕЛ)

### Standard

The bibliography is formatted per **ДСТУ 8302:2015** "Бібліографічне посилання". The methodology directs to a sample on pages 122–124 of Avramenko et al. (2018) for further examples.

### Minimum count and target

- Methodology minimum: **10 sources**.
- Master's-level target: **25–30 sources**.
- Composition target: ~70% English-language Q1/Q2 journal articles from 2020–2026, ~20% Ukrainian-language sources, ~10% official reports and standards.

### Ordering

Two acceptable orders, pick one:

- **Alphabetical** by first author's surname (Cyrillic first, then Latin).
- **By order of first appearance** in the text.

The order chosen must be consistent across the entire list.

### Formatting

- Each entry is a numbered item (1, 2, 3, ...) without decoration.
- Single line spacing within an entry, blank line between entries (or hanging indent).
- Times New Roman Cyr 14 pt, same as body.

### Standard ДСТУ 8302:2015 patterns

#### Book by single author

```
Прізвище І. П. Назва твору. Місце видання : Назва видавництва, Рік. К-сть с.
```

Example:
```
Шевченко Т. Г. Кобзар. Київ : Наукова думка, 2012. 450 с.
```

#### Article in a journal or proceedings

```
Прізвище І. П. Назва статті. Назва журналу. Рік. Т. X, № Y. С. AA–BB.
```

Example:
```
Saaty T. L. Decision making with the analytic hierarchy process. International Journal of Services Sciences. 2008. Vol. 1, № 1. P. 83–98.
```

#### Conference paper

```
Прізвище І. П. Назва доповіді. Назва конференції : тези доп., м. Місто, дата. Місто : видавництво, Рік. С. AA–BB.
```

#### Electronic resource

```
Назва ресурсу. URL: посилання (дата звернення: дд.мм.рррр).
```

Example:
```
IEA. Global EV Outlook 2025. URL: https://www.iea.org/reports/global-ev-outlook-2025 (дата звернення: 07.05.2026).
```

#### Standard

```
ДСТУ 8302:2015. Інформація та документація. Бібліографічне посилання. Загальні положення та правила складання. [Чинний від 2016-07-01]. Київ : ДП «УкрНДНЦ», 2016. 17 с.
```

### Authors

- Up to three authors: list all with initials before surname.
- Four or more authors: list first three, then `та ін.` (Ukrainian) or `et al.` (English source).
- Initials with non-breaking spaces between them: `Saaty T. L.`, not `Saaty T.L.`.

### Page ranges and volumes

- Pages: `С.` (Ukrainian) or `P.` (English) followed by number.
- Volume: `Т.` (Ukrainian) or `Vol.` (English).
- Issue: `№` followed by number.
- En-dash for ranges: `С. 12–34` (not hyphen `12-34`).

### English-language sources

For English-language sources you may keep the original publication metadata language. Do not translate the title.

### Common errors

| ❌ | ✅ |
|---|---|
| `Saaty T.L.` | `Saaty T. L.` (non-breaking spaces) |
| `С.12-34` | `С. 12–34` (space and en-dash) |
| Period missing after surname | `Шевченко Т. Г.` (period after each initial) |
| Comma between place and publisher | Colon: `Київ : Наукова думка` |
| URL without access date | `(дата звернення: дд.мм.рррр)` |

## Appendices (ДОДАТКИ)

### Separator sheet

Before the first appendix, insert a **blank sheet of paper** with the word **ДОДАТКИ** centered, in uppercase, large font. **No page number on this sheet** (but it counts in the continuous numbering — the next sheet has the next number).

### Appendix lettering — coursework-specific

The list of forbidden letters in the coursework methodology is:

**Forbidden:** Є, С, З, Ї, Е, І, Й, О, Ч, Ь.

**Allowed sequence:** А, Б, В, Г, Д, **Ж**, **К**, Л, М, Н, П, Р, Т, У, Ф, Х, Ш, Щ, Ю, Я.

**Important:** this list differs from the МДР list (which forbids Е, Є, З, І, Ї, Й, О, Ч, Щ, Ь). For this coursework follow the **coursework-specific list**.

After Д the next allowed letter is Ж. After Ж the next is К. After Н the next is П. Memorize this transition pattern.

### Appendix heading

Each appendix starts on a new page. The heading format:

```
Додаток А
Лістинг ключових модулів backend
```

Two-line layout:
- Line 1: «Додаток А» (or whatever letter), centered.
- Line 2: name of the appendix, lowercase except first letter, centered, bold.

If a single appendix continues onto a second sheet:
```
Продовження додатку А
```

### Numbering inside appendices

Figures, tables, formulas inside an appendix use the appendix letter as prefix:
- `Рис. А.1` — first figure of Appendix А;
- `Таблиця Б.2` — second table of Appendix Б;
- `(Г.1)` — first formula of Appendix Г.

If the appendix is divided into its own subsections, the section number gets the letter prefix: `А.1`, `А.2`, etc.

### Page numbering across appendices

Page numbers continue through appendices in the document's overall sequence. The separator sheet has no number printed (but the position counts in the sequence).

### Code listings in appendices

Source code in appendices follows different formatting from body text:

- Font: **Courier New 10 pt** or Arial 10 pt (per the formatting requirements document).
- Line spacing: single.
- The methodology requires that the **full code listing be in appendices, never in the main text**. In Section 3.1 only short code fragments (10–25 lines) are allowed for illustrating the most interesting algorithmic points.

### Appendix composition

The list of appendices and the content assigned to each is defined in `docs/СТРУКТУРА_КУРСОВОЇ.md` — consult that file. This reference covers only the formatting rules for appendices (letters, numbering, headings), not which appendices exist.

### Multi-document appendices

If an appendix contains an existing document with its own internal numbering (e.g., a copy of a technical assignment or a contract), the methodology allows:

- Place the document as-is.
- The label «Додаток _» appears in the top-right corner of the document's first page (if there's room) on a second line with the appendix name.
- If there is no room on the document, place a separator sheet before it with «Додаток _» centered, then the document follows with its own internal pagination but with the work's continuous page numbers added to the bottom-right corner.

## Common errors in appendices

| ❌ | ✅ |
|---|---|
| `Додаток Е` | `Додаток Ж` (Е is forbidden in coursework) |
| `Додаток З` | `Додаток К` (З is forbidden) |
| `Додаток І` | `Додаток К` (І is forbidden) |
| Code listing in main text section 3.1 | Code listing in Додаток А; only fragments in 3.1 |
| Code in Times New Roman 14 | Code in Courier New 10 |
| Page number on the «ДОДАТКИ» separator sheet | No page number on the separator sheet |
| Continuous figure numbering (`Рис. 4.1` for the first figure of an appendix) | `Рис. А.1` (letter prefix) |
| Each appendix continuing the previous one's pagination | Each appendix starts on a new page |
| Two appendices with the same purpose | Each appendix has a unique purpose and title |

## Cross-reference cheatsheet — appendix letter sequence

```
Position | Letter | Allowed?
---------|--------|----------
1        | А      | ✓
2        | Б      | ✓
3        | В      | ✓
4        | Г      | ✓
5        | Д      | ✓
6        | Е      | ✗ FORBID
7        | Є      | ✗ FORBID
8        | Ж      | ✓
9        | З      | ✗ FORBID
10       | И      | ✗ FORBID (І, Й also forbidden)
11       | К      | ✓
12       | Л      | ✓
...
```

When jumping over forbidden letters, no comment or note is required — simply use the next allowed letter. The purpose assigned to each appendix is defined in `docs/СТРУКТУРА_КУРСОВОЇ.md`, not here.
